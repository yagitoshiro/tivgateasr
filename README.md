tivgateasr
==========

vGate ASR module for Titanium(Android)

Usage
==========
See example/app.js

Install
==========
First subscribe to docomo developer support and get SDK and API key. Put libFSR.so into lib/armeabi-v7a/. Currently only ARM processers are supported (by vendor).
